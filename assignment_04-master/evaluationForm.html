Introduction

The Accessibility and Usability of a website are significant features to show the quality of a website. Having taken into consideration that, This website is designed by showing the functionality of the website to all users as well as ensuring the accessibility of content to users with certain disabilities .

The accessibility and the usability aspects of the website is to include alternate text descriptions to non-text content such as images. The appearance, surroundings and colors are involved. The time limitation prevent to create an accessible and usable content, some principle guidelines is not met. In this evaluation report, the different criterion unmet with methods on how the website can improve to fulfill the unmatched criteria is discussed.

Accessibility Features

In this section,the key factors that makes the website accessible will be issued and some aspects of the website that has a potential negative impact on the accessibility will be evaluated in the lights of W C A G guidelines. Furthermore, You can see and compare each features as a table. Appendix A

Firstly, the accessibility of the website provides alternate text information to image files on the page. These alternate texts were implemented through the usage of alt' attributes within an 'img' tag. Moreover, these alternate tags contain a brief description of the image, ensuring to use words involving colors and surroundings to enable the text to equally convey an information as the actually image does. This is highly applicable to the W C A G guideline 1.1 criteria 1.1.1 where is mentioned that " A website should provide text alternatives for non-text content, serving an equal purpose in conveying information. To sum up, W C A G guideline is highly relevant to the website in a positive aspect; allowing visually impaired users to perceive the image being displayed on the page.

Secondly, the website distinguishes the content from the background. This was mainly done by the use of color as the only visual mean of conveying information to the user. For instance, the website has links in the body and footer that involves colors to indicate whether the link has been visited or unvisited as well as using bolded text to convey importance of information and headers. this is highly relevant to the W C A G guideline 1.4, criteria 1.4.1. Yet, this can be issue for the color blind people.

Finally, the navigability of the website in W C A G guideline 2.4 provide a great significance to the website. The success criteria 2.4.1, 2.4.2 and 2.4.4 are met. This is done by including descriptive links that give to the user its purpose just by reading the link text(2.4.4). Headers and titles are able to give the overview of the topic of each content(2.4.2) and they include links in the navigation bar which bypass other unwanted blocks to lead the user through the place of the each content(2.4.1).
